
# RPSLS - Rock Paper Scissors Lizard Spock

## Author
**Motz, Bénédicte**

## Description
This is a real-time multiplayer game based on the extended Rock Paper Scissors Lizard Spock rules. It is built using **Next.js** for the frontend and **Express + Socket.IO** for the backend. Players connect, select their moves, and receive instant feedback on the outcome of each round.

## Features
- Real-time communication using Socket.IO
- Modular server-side handlers for connection, disconnection, moves, and rounds
- Frontend interface built with React and Next.js
- Support for multiple connections per player
- Rule-based outcome logic for RPSLS

## Setup Instructions

# install dependencies
npm install

# start backend
npm run server

# start frontend
npm run dev 

Open app in browser and go to http://localhost:3000
